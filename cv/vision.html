Interests & Vision

At the end of the day and the end of my life I want to be able to say that I contributed more than I criticized.

Brené Brown

I am interested in languages, frameworks & tools that encourage the development of scalable software.

Scalability is the cornerstone of all application development aspects. It shapes how users interact with a computer:

  • The choice of programming language determines how developers can express their intents.
  • Build & test automation tames complexity.
  • Efficient data structures and performant algorithms are crucial for meeting big data demands. They are the foundation for interactive user interfaces.
  • Reactive architectures encourage systems that are robust, resilient, and flexible enough to meet modern requirements.
  • Responsive user interfaces guarantee scalability across the range of end-user devices.

Project Pipeline

Bacterial Cell Segmentation & Lineage Visualization — FHNW & ETH/Eawag

The Molecular Microbial Ecology research group at ETH/Eawag focuses on the ecology and evolution of bacteria. The group uses a semi-automated program pipeline to analyze vast amounts of microscope images. Certain steps in this program pipeline are not optimized for efficient manual analysis, and are thereby limiting hypothesis generation.

A new GUI with a more efficient interaction concept and tailor-made visualizations will mitigate this bottleneck.

EduNaT Project Nr. 8 — FHNW

When teaching computer programming, live coding demonstrations are very valuable for students. After a lecture, students might want to reproduce a live coding session at home, at their own pace. However, traditional video recordings are not powerful enough to capture the complete programming process. Specifically, students should also be able to actually work with the code as it evolved during the demonstration.

A new tool will allow to capture and reproduce live coding sessions on the level of source code changes.
