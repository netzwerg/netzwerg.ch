Graphical User Interfaces (GUIs)

It seems that perfection is attained not when there is nothing more to add, but when there is nothing more to remove.

Antoine de Saint Exupéry

This famous quote is especially true for graphical user interfaces (GUIs). Ever since I started programming, I always had a passion for GUIs and taming complexity. While computers give super powers to programmers, GUIs give super powers to non-programmers.

As a scientist, I am particularly interested in visualizing big, complex data. Extracting meaningful information and making it accessible to humans is very challenging and extremely rewarding.

While I love the sheer unlimited possibilities of fat clients, the ubiquitous availability of browsers makes web applications a good choice, too. On a technical level, the latest trend of transpiling high-level languages such as Elm or Scala to Javascript makes the browser a truly attractive target platform.

In addition to my background in creating scientific fat clients, I keep building websites for friends & family. This allows me to hone my skills and keep up with latest trends. I am thrilled how libraries like d3.js take my graphical abilities to a whole new level.

Selected Applications

xPlan — FHNW & University Hospitals Bern/Basel/Zürich

xPlan is a JavaFX application for hospital infrastructure planning and process optimization. Data from various hospital information systems was collected, enriched with topographical and organizational data, and integrated into a coherent data store. xPlan offers various interactive views onto this data store.
Contribution: I implemented an interactive 3D view, which allows tracking and quantification of patient flows through the hospital buildings.

Green Radiology — FHNW & University Hospital Basel

Green Radiology is a JavaFX time series investigation tool which allows analyzing the carbon footprint of large-scale radiology devices (MRI, CT). It can be used for any kind of sequentially measured data. Tailor-made optimizations allow loading huge data volumes, while still offering responsive user interactions and a small memory footprint.
Contribution: I implemented the majority of the graphical user interface as well as the data models. The core data structures are based on my open-source paleo library.

Genedata Screener — Genedata AG

Genedata Screener is a comprehensive software solution that captures, analyzes, and manages all types of screening data. The product suite has a track record of 13 major releases and is used by big pharma companies throughout the world (Amgen, AstraZeneca, Bayer, Novartis, Pfizer).
Contribution: I was a long-term member of Screener's core development team, responsible for the design, architecture, implementation & maintenance of all application components (Java front- & back-end, Python service APIs).

Edoras One — Edorasware AG

Edoras One is a case and business process management suite based on the Activiti BPM Platform.
Contribution: I was a member of Edorasware's core development team, responsible for the design, architecture & implementation of the complete software suite (Angular web-client, REST API, embeddable Java process engine).
