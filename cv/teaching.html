Selected Teaching Activities

Programming II — Life Science Technologies, FHNW

I teach this module in collaboration with Prof. Dr. Dominique Brodbeck. The course covers advanced programming principles, using the Java language.

For the current semester, I completely restructured the course material and made it available publicly. The material is optimized for a flipped classroom setting: Students get a summary of the most important course topics which they research on their own. A set of well-crafted exercises helps them track their own progress and evaluate their learning. Exercises are set up in a test-driven fashion: A suite of automated tests expresses each exercise's requirements. Initially, all tests fail and students are required to work on their tasks until all tests pass. Automated tests provide immediate feedback and can be a great motivation.

The course is rounded off with a final project, in which students are asked to implement a full-fledged JavaFX application.

In-class lectures are spent on live programming, discussions, and personal code reviews.

Git Workshop — Basler Versicherungen

Git is a free and open source version control system which is known to have a steep learning curve. During my work at Basler Versicherungen, I was responsible for migrating all software development projects from CVS to Git. Theoretically, I only had a technical role in this migration. Thanks to my flair of explaining difficult things in simple words, I ended up teaching Git workshops across the complete software department. I enjoyed these workshops a lot, there's nothing as rewarding as hearing a "Thank you, finally I get it!" from a participant who previously felt lost and confused.
