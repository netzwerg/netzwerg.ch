Application Programming Interfaces (APIs)

Human-Computer Interaction researches the design and use of computer technology, focusing on the interfaces between people and computers.

Wikipedia

Software developers are humans, too. A programming language is a user interface for developers. The way programmers can interact with a language is defined by its application programming interface (API).

As a software developer, I am passionate about using programming languages with well designed APIs. I know Java inside out and enjoy using it every day. A few years back, I shifted my passion to Scala, a language with a very clean syntax and powerful functional programming paradigms.

When writing software, programmers create reusable pieces of software called libraries. Solving a challenging software problem is one thing; sharing a reusable library with other developers is a whole other level. As a maintainer and contributor to many open-source projects, I embrace this challenge on a daily basis. Designing good library APIs, contributing features or bug fixes, and collaborating internationally makes the world a better place — one commit at a time.

Selected Open-Source Projects

Javaslang — Contributor

Java is one of the most popular programming languages in the world. Yet, some of its APIs and data structures are outdated. Javaslang is a library that complements the standard Java functionality. It provides efficient data structures and modern, functional constructs.

I contributed an optimized search algorithm for one of Javaslang's core data structures (see pull request).

Gradle Release Plugin — Owner/Maintainer

Gradle is a popular build tool. Out-of-the-box, it is missing the functionality to create semantically versioned releases in an automated fashion. My Gradle Release Plugin provides this missing functionality. Since its publication in Dec 2015, the plugin has been downloaded almost 10k times.

Paleo Data Frames — Owner/Maintainer

The concept of a data frame comes from the world of statistical software. My paleo library brings this powerful concept to the Java ecosystem. Paleo offers a fully type-safe, programmer-friendly API, while exhibiting a very small memory footprint.

The paleo library represents the foundation of our current big data visualization projects (see Green Radiology).
